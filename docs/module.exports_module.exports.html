Class: exports

exports(orientationopt, unitopt, sizeopt, PDFAopt, posXopt, posYopt)

new exports(orientationopt, unitopt, sizeopt, PDFAopt, posXopt, posYopt)

This is a constructor function for a JavaScript class that initializes various properties and settings for creating PDF documents.
Parameters:
Name Type Attributes Default Description
orientation <optional>
 P The orientation of the document, either 'P' for portrait or 'L' for landscape.
unit <optional>
 mm The unit parameter determines the unit of measurement used for the document. It can be set to 'pt' (points), 'mm' (millimeters), 'cm' (centimeters), or 'in' (inches).
size <optional>
 A4 The size parameter determines the size of the document. It can be one of the following values: 'a3', 'a4', 'a5', 'letter', or 'legal'. The default value is 'A4'.
PDFA <optional>
 false A boolean value indicating whether the PDF should be generated in PDF/A format.
posX <optional>
 1 The posX parameter represents the X position of the starting point for the labels. It determines the horizontal position of the first label on the page.
posY <optional>
 1 The `posY` parameter represents the initial vertical position of the document or label. It determines the starting point for drawing content on the page.
